A morphology-based (etymology-based) English vocabulary learning tool designed to help learners deeply understand word construction logic through **Prefix**, **Root**, and **Suffix** decomposition. Combined with **Scenario Examples** and **Language Levels** (C1), it enables efficient and deep vocabulary retention and application.

## ğŸŒŸ Features

- **Morphological Breakdown**: Decomposes words into prefixes, roots, and suffixes to intuitively demonstrate word construction logic.
- **Multi-dimensional Indexing**: Supports exploring vocabulary by prefix, root, suffix, and phrase dimensions.
- **Leveled Learning**: Covers core vocabulary for different language levels such as C1.
- **Scenario Examples**: Provides rich real-world scenario examples to help master usage in context.
- **Pronunciation Support**: Integrated IPA symbols and UK/US pronunciation (resources required).
- **Recite Mode**: Supports hiding definitions for self-testing.
